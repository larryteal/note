## 开发环境
### vue-cli 工具
  * npm i -g @vue/cli  # 新版
  * npm i -g vue-cli  # 旧版
  * webpack 
### 调试
  * Chrome 调试多条命令一起执行
  * 可以先使用 shift + enter 换行 后再 enter 一起执行
  * 可以把Vue实例赋值给一个变量，然后在Chrome 调试时可以直接使用这个变量，也能通过这个实例操作变量做一些改变。
  * 可以在代码中写 debugger 关键字，代码运行到debugger时就会暂停类似打断点
